## File Validation

### Supported Formats
- **CSV**: Comma-separated values
- **JSON**: Structured data format
- **Max Size**: 500MB per file
- **Max Rows**: 50,000 rows

### Required Columns
- **Text Column**: Content for sentiment analysis
- **ID Column**: Unique identifier (optional)
- **Date Column**: Timestamp (optional)

### Error States
- **File too large**: "File exceeds 500MB limit"
- **Invalid format**: "Please upload CSV or JSON files only"
- **Missing columns**: "Required text column not found"
- **Processing error**: "Failed to process file. Please try again"
