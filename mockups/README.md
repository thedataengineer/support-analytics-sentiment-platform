# Sentiment Platform Mockups

## Design System

### Color Palette
- **Primary**: #1976d2 (Material Blue)
- **Secondary**: #dc004e (Material Pink)
- **Success**: #2e7d32 (Green)
- **Warning**: #ed6c02 (Orange)
- **Error**: #d32f2f (Red)
- **Background**: #f5f5f5 (Light Gray)
- **Surface**: #ffffff (White)

### Typography
- **Headers**: Roboto, 24px/32px/20px
- **Body**: Roboto, 16px/14px
- **Captions**: Roboto, 12px

### Components
- Material-UI components
- Recharts for data visualization
- Plotly for advanced analytics

## Mockup Files

1. **dashboard-mockup.md** - Main dashboard with sentiment overview
2. **upload-mockup.md** - CSV upload interface
3. **analytics-mockup.md** - Advanced analytics dashboard
4. **ticket-trajectory-mockup.md** - Individual ticket analysis
5. **login-mockup.md** - Authentication interface

## Layout Principles

- **Responsive**: Mobile-first design
- **Accessibility**: WCAG 2.1 AA compliance
- **Performance**: Lazy loading for large datasets
- **UX**: Minimal clicks, clear navigation
